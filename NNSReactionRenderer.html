Reaction Renderer

Preview status: Up to date
Format example: r2_2, 100, 1, COONa, 1, Cu2+, k1, 1, COOCu+, 1, Na+
Species are rendered as normal text (not LaTeX). Lines starting with # are comments: rendered as text, not parsed as equations, and not equation-numbered.
Load input TXT by clicking Load nested TXT in the left sidebar, select all related TXT files at once, then choose the main TXT and click Load selected main file. The main file is the file that starts to refer other files but no files refer it. It is the parent of all children files.
